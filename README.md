## Get Started

### Installation

```
$ npm install metagpt
```

### Configuration

create `.env` file:

```
OPENAI_KEY=sk-xxxx
```

(Currently just use OpenAI api as LLM response client)

### Usage

Could refer the examples in `examples/**`.
